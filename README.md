## Descripción
Crear una app web bajo los principios de SPA (Single page aplication).

Para ello debes entender y aplicar los siguientes conceptos.

* Templating
* Routing (URLs)
* DOM & Event Handling (State)
* Patrón de arquitectura de software (MV*)
